# sheltersNorwayMobile

<p> Mobile application that displays public bombshelters located around Norway. Each shelter contains data about location, google maps chart, name, available spots, year and much more.</p>
<p>This application is a mobile makeover of the <a href="https://github.com/sanderhelleso/sheltersNorway">web version</a>.It trumps the web version in speed, functionality and UX and is recommended as a companion to the webversion if internet is not accesible.</p>

## Includes

<ul>
  <li>Find your closest shelter from your location</li>
  <li>Search for specific shelter and sort by a wide variety of criterias</li>
  <li>See all available shelters registered</li>
  <li>Interactiv map for all shelters, easy to learn and discover</li>
  <li>Own details page for every shelter</li>
  <li>Quicksearch by storing previous search and caching them</li>
  <li>All but the map tiles, works offline!</li>
</ul>

## Why

<p>Alot of requests has been made for a mobile app that works offline</p>
